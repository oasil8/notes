# Windows 단축키 마스터 가이드

---

## 1. 시스템 제어 및 기본 설정

가장 자주 사용하는 윈도우 핵심 기능입니다.

| 단축키 | 기능 |
|--------|------|
| `Win + E` | 파일 탐색기 열기 |
| `Win + I` | 윈도우 설정 창 열기 |
| `Win + R` | 실행 창 열기 (명령어 입력 시 사용) |
| `Win + L` | 화면 즉시 잠금 (보안) |
| `Win + V` | 클립보드 기록 (복사했던 목록 확인) |
| `Win + .` | 이모지 및 특수 기호 입력창 |
| `Ctrl + Shift + Esc` | 작업 관리자 즉시 실행 |
| `Win + Shift + S` | 화면 캡처 |

---

## 2. 창 관리 및 멀티태스킹

여러 창을 띄워놓고 작업할 때 생산성을 극대화합니다.

| 단축키 | 기능 |
|--------|------|
| `Alt + Tab` | 열려 있는 창 간 전환 |
| `Win + D` | 모든 창 최소화 / 다시 누르면 복원 (바탕화면 보기) |
| `Win + ← / →` | 화면 2분할 (왼쪽/오른쪽 붙이기) |
| `Win + ↑ / ↓` | 창 최대화 / 이전 크기 복원 / 최소화 |
| `Win + Z` | 창 분할 레이아웃 선택 (**Windows 11 전용**) |
| `Win + (1~0)` | 작업 표시줄 프로그램 실행 (`Shift` 추가 시 새 창으로 실행) |
| `Ctrl + W` | 현재 활성화된 창이나 탭 닫기 |

---

## 3. 가상 데스크톱 (작업 공간 분리)

모니터 하나를 여러 개처럼 사용하고 싶을 때 필수입니다.

| 단축키 | 기능 |
|--------|------|
| `Win + Ctrl + D` | 새 가상 데스크톱 생성 |
| `Win + Ctrl + ← / →` | 생성된 데스크톱 간 화면 전환 |
| `Win + Ctrl + F4` | 현재 가상 데스크톱 닫기 (창은 이전 화면으로 이동) |

---

## 4. 파일 및 폴더 관리

마우스 우클릭 시간을 줄여주는 단축키들입니다.

| 단축키 | 기능 |
|--------|------|
| `F2` | 파일/폴더 이름 바꾸기 |
| `Alt + Enter` | 선택한 파일의 속성(정보) 보기 |
| `Ctrl + Shift + N` | 새 폴더 만들기 |
| `Shift + Del` | 휴지통을 거치지 않고 영구 삭제 |
| `Win + Shift + S` | 화면 캡처 (영역 지정 가능) |

---

## 5. 문서 작업 및 텍스트 편집

타이핑 속도와 편집 효율을 높여줍니다.

| 단축키 | 기능 |
|--------|------|
| `Ctrl + C / A / V / X` | 복사 / 전체 선택 / 붙여넣기 / 잘라내기 |
| `Ctrl + Z / Y` | 실행 취소 / 다시 실행 |
| `Ctrl + S` | 작업 내용 저장 |
| `Ctrl + B` | 선택한 텍스트 굵게(Bold) 적용 |
| `Ctrl + Backspace` | 단어 단위로 한 번에 지우기 |
| `Ctrl + Shift + 방향키` | 단어 단위로 텍스트 블록 지정 |
| `Tab` | 들여쓰기 (공백 4칸 수준) |

---

## 6. 웹 브라우저 및 메신저 (크롬, 카톡 등)

| 단축키 | 기능 |
|--------|------|
| `Ctrl + T` | 새 탭 만들기 |
| `Ctrl + Shift + T` | 실수로 닫은 탭 다시 살리기 |
| `Ctrl + Tab` | 브라우저 내 탭 전환 (`Shift` 추가 시 역방향) |
| `Ctrl + L` | 주소창으로 바로 이동 (검색어 입력) |
| `Ctrl + H` | 방문 기록(History) 열기 |
| `Ctrl + D` | 현재 페이지 북마크 추가 |
| `Ctrl + 마우스 휠` | 화면 확대 및 축소 (또는 `Ctrl + - / +`) |
| `Shift + Enter` | (카카오톡 등) 전송하지 않고 줄바꿈 |
