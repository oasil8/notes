## 2. 로컬 AI 구축 전략
 
### 2.1 검토된 스택
 
| 컴포넌트 | 역할 |
|---|---|
| Ollama | 로컬 모델(Llama, DeepSeek 등) 다운로드 및 터미널 실행 |
| Docker + Open WebUI | ChatGPT 유사 웹 UI, 컨테이너 기반 안정 구동 |
| Continue 플러그인 | VS Code / JetBrains IDE 연동 로컬 코파일럿 |
 
### 2.2 하드웨어 요건
- 한국어 성능 및 정교한 추론: **최소 14B 파라미터 이상** 필요
- 쾌적한 구동: 고용량 VRAM 필수 (RTX 5090급 벤치마크 참고)
- 맥 미니 M4 Pro 24GB 기준 실행 가능 모델:
| 모델 | 속도 | 주요 용도 |
|---|---|---|
| Llama 3.1 8B | ~80 tok/s | 빠른 Q&A |
| Qwen2.5-Coder 14B | ~45 tok/s | 코드 생성, Isaac Sim 스크립트 |
| DeepSeek-R1 32B | ~20 tok/s | 논리 추론, 논문 분석 |
