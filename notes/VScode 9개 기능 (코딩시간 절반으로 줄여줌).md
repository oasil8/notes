---
코딩시간을 절반으로 줄여주는 VSCode 9개 기능
---
1. Go to Definition (정의로 이동)

함수/변수가 어떻게 구현되어 있는지 확인할 때:

- F12 — 해당 함수/변수 정의로 바로 이동
- Alt+F12 — 현재 위치에서 팝업으로 미리보기 (Peek)

CSS/HTML에도 적용하려면 익스텐션 설치 필요:

- CSS Peek
- HTML CSS Support

---

2. Refactoring (리팩토링)

코드를 선택(하이라이트)한 후 사용:

┌─────────────────────────┬─────────────────────────────────────────────────────────────────────┐
│          기능           │                                        │
├─────────────────────────┼─────────────────────────────────────────────────────────────────────┤
│ 코드를 함수로 추출      │ 선택 후 Ctrl+Shi                       │
├─────────────────────────┼─────────────────────────────────────────────────────────────────────┤
│ 코드를 변수로 추출      │ 선택 후 Ctrl+Shift+R → "상수 추출" 선택                             │
├─────────────────────────┼─────────────────────────────────────────────────────────────────────┤
│ 코드를 다른 파일로 분리 │ 선택 후 Ctrl+Shift+R → "Move to new file" 또는 왼쪽에 전구(💡) 클릭 │
└─────────────────────────┴─────────────────────────────────────────────────────────────────────┘

---

3. Rename Symbol (심볼 일괄 이름 변경)

변수명/함수명을 모든 파일에서 안전하게 바꿀 때:

- F2 — 해당 심볼이 쓰인 모든 곳을 자동으로
- Ctrl+H (Find & Replace)보다 스마트하고 안전함 (스코프 인식)

---

4. Snippets (코드 스니펫 자동완성)

언어/프레임워크별 스니펫 익스텐션 설치 추천:

- React → VS Code ES7+ React/Redux/React-Native snippets 설치
  - imr + Tab → React import
  - imrs + Tab → useState import
  - rfce + Tab → React 함수형 컴포넌트 생성
- Vue → Vue 관련 스니펫 설치 시 .vue 파일 템플릿 자동완성

---

5. AI 자동완성 (Tabnine)

- Tabnine 익스텐션 설치 시 AI가 다음 코드를
- fo → for 반복문 전체 자동완성
- co → counter 변수 자동 완성
- 단점: 유료 (월 약 1만원)

---

6. Debugger (디버거)

복잡한 반복문이나 로직의 변수 값을 추적할 때:

1. 코드 줄 왼쪽 여백 클릭 → 빨간 점(브레이크포인트) 설정
2. F5 → 디버그 모드로 실행
3. 실행이 브레이크포인트에서 멈춤 → 왼쪽 패
4. 디버그 콘솔에서 변수명 직접 입력해서 현재 값 확인 가능
5. 단계 진행 버튼으로 한 줄씩 실행하며 추적

---

7. Multi Cursor (멀티 커서)

여러 곳을 동시에 편집할 때:

- Alt + Click — 원하는 위치에 커서 여러 개 동시 추가
- 이후 타이핑하면 모든 커서 위치에서 동시에 입력됨

HTML:
div.container>p.title*3   → Tab → div 안에 클래스 title인 p 태그 3개 생성

CSS:

┌──────┬─────────────────┐
│ 입력 │    Tab 결과     │
├──────┼─────────────────┤
│ dn   │ display: none;  │
├──────┼─────────────────┤
│ df   │ display: flex;  │
├──────┼─────────────────┤
│ mcv  │ margin: (value) │
└──────┴─────────────────┘

---

9. 키보드 단축키 (마우스 없이 이동)

┌────────────────────────────────┬────────────────────────────────────────┐
│              기능              │                 단축키                 │

│ 단어 단위로 이동               │ Ctrl + ← / →                           │
├────────────────────────────────┼────────────────────────────────────────┤
│ 현재 줄 복사 (아래에 붙여넣기) │ Alt + Shift + ↓                        │
├────────────────────────────────┼───────────┤
│ 파일 빠른 검색 & 이동          │ Ctrl + P                               │
├────────────────────────────────┼────────────────────────────────────────┤
│ 터미널 열기/닫기               │ Ctrl + ` (물결 표시)                   │
├────────────────────────────────┼────────────────────────────────────────┤

└────────────────────────────────┴────────────────────────────────────────┘
